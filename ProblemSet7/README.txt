HOW TO INPUT PARAMETERS INO THE PROGRAM

OPEN TEXT FILE NAMED : parameters.txt

EACH LINE IS COMMENTED. NO NEED TO DELETE THE COMMENTS. ONLY REPLACE THE PARAMETERS (NUMBERS) IN THE TEXT FILE TO THE REQUIRED TEST CASES. DO NOT CHANGE THE FORMAT OF THE TEXT FILE, OR ERRORS MAY OCCUR DURING PARSING.


BEFORE RUNNING THE EXECUTABLE a.out, please include the source data files in the same folder. By default, the executable is for face.txt

To prevent overwriting the submitted output, the output from the above executable file will be in “output.wrl”
